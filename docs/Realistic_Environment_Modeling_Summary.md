### **2. IEEE 802.15.4链路质量评估**
```python
# RSSI到LQI的标准映射
lqi = (rssi - sensitivity_threshold) / rssi_range * 255

# 基于过渡区域理论的PDR计算
if rssi > -70: pdr = 0.99
elif rssi > -80: pdr = 0.5 + 0.49 * (rssi + 80) / 10
else: pdr = max(0.0, (rssi + 85) / 5 * 0.5)
```

